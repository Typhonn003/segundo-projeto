Atributo "target" dentro da tag <a>

<a href="https://www.link.com" target="Valores">Palavra clicavel para o link</a>

Valores

  • _self vai abrir o link na janela atual (padrão)
  • _blank vai abrir o link em uma nova janela em branco

Exemplos de como vai ficar

  • Abrir na mesma janela
  • Abrir em uma nova janela

Atributo "rel" dentro da tag <a>

<a href="https://www.link.com" target="_self" rel="Valores">Palavra clicavel para o link</a>

Valores

  • next indica que o link é para a próxima parte do documento atual
  • prev indica que o link é para a parte anterior do documento atual
  • author indica que é um link para o site do autor do artico atual
  • external indica que é um link para outro site que não faz parte do site atual
  • nofollow indica que é um link para um site não endossado, como um link pago

Criando links de download com a tag <a>

<a href="Link ou caminho para o item do download" download="Nome do arquivo com a extensão" type="Media types">Palavra clicavel para o link</a>

O atributo download dentro da tag <a> deve ter em seu valor o nome do arquivo de download junto com sua extensão.
Exemplo: download="meulivro.pdf"

O atributo type dentro da tag <a> deve ter em seu valor o tipo de arquivo que está sendo baixado, como o exemplo a baixo.

  • application/zip indica que é um aplicativo com a extensão zip
  • video/mp4 indica que é um video no formato mp4
  • image/png indica que é uma imagem no formato png

Atributo "hreflang" dentro da tag <a>

Exemplo do código, Jocsã comentou que é utilizado apenas em casos específicos para otimizar o bot do google, caso o link esteja em um site brasileiro e leve para outro site brasileiro, não tem necessidade de utilizar o atributo hreflang.

<a href="https://www.link.com" hreflang="idioma de origem do site">Palavra clicavel para o link</a>

Dentro do atributo hreflang devo indicar o idioma do site para qual o link irá redirecionar, isso permite avisar ao navegador e softwares de tradução como lidar caso o visitante opte por traduzir automaticamente os conteúdos.
